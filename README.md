# Pixnights
## Dependendcy
SFML 2.6.1 (Can be installed for Visual Studio 2022 with NuGet)

## Build (Visual Studio 2022)
```powershell
cmake -G "Visual Studio 17 2022" -A x64 .
```
and you will get a solution file to open with Visual Studio 2022.
